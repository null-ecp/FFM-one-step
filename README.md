## 一键搭建FFM脚本

> 在服务器上输入

```bash
#先安装下git
#这是centos安装git
yum install git
#这是ubuntu安装git
apt-get update
apt-get install git
#这是Fedora
dnf install git
#获取脚本
git clone https://github.com/null-ecp/FFM-one-step.git && source ./FFM-one-step/FFM.sh
```
